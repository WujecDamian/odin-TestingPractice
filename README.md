# Odin Project - Testing Practice

Solutions and practice exercises for the **Testing Practice** project from [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-testing-practice) (JavaScript path).

## About

This repository contains my work on unit testing in JavaScript using **Jest**.  
The project focuses on writing clean, testable code and creating comprehensive test suites for various functions (string manipulation, calculators, Caesar cipher, etc.).

## Main Technologies

- JavaScript (ES6+)
- Jest (testing framework)
- Webpack (module bundler)
- Babel (JavaScript transpiler)
